To run
1. In the command line, type in: npm install
2. Run ddl.sql to create all tables
3. Input the correct db_username, db_password, and db_name into the server.js and database-initializer.js files
4. Initialize databse with (this is found in the dbinit directory): node database-initializer.js 
5. Run the server: noder server.js
6. Access the web application: http://localhost:3000
7. Register as a user to test all functionalities

As admin
1. Access web application: http://localhost:3000/admin
2. Log in as admin with
	Username: admin
	Password: admin
